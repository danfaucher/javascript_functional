Functions as First-Class Objects

Saying objects are first-class means you can do with functions what you can do with other objects.You can pass functions as parameters to other functions or return a function as a result of a function call.

Recursion

A function can call itself and when that call is done work with the result.

Example

//factorial.js
        function fact(n) {
            if (n===0) {
                return 1;
            }else{
                return n * fact(n -1);
            }
        }

        console.log(fact(5)); // 120

Closures

Closures are a way to implement data hiding. A function can refer to its local variables and eveything outside of the function
